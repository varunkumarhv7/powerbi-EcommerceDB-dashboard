# 📊 E-Commerce Sales Dashboard (Power BI)

## 🔹 Overview
This project is a Power BI dashboard analyzing e-commerce sales data.

## 🔹 Key Metrics
- Total Revenue
- Total Orders
- Total Quantity Sold
- Average Order Value

## 🔹 Visuals
- Sales by Product
- Top 5 Products
- Interactive Filters (Slicers)

## 🔹 Tools Used
- Power BI
- DAX (Calculated Columns & Measures)
- Data Modeling

## 🔹 Insights
- Identified top-performing products
- Analyzed revenue distribution
- Built interactive dashboard for decision-making
